# TO_DO_LIST

Aplicação simples em Python para gerenciamento de uma lista de compras via terminal. O foco é praticar manipulação de listas, entrada de dados, estrutura de repetição e tratamento de erros. O projeto segue uma abordagem direta e objetiva.

## Recursos

* Inserção de itens
* Remoção por número
* Listagem numerada a partir de 1
* Prevenção de falhas por índices inválidos
* Limpeza do console para melhor leitura

## Execução

```
python TO_DO_LIST.py
```

## Estrutura do código

* enumerate é utilizado com início em 1 para facilitar a leitura do usuário
* try/except garante que entradas incorretas não interrompam o programa
* os.system é usado apenas para manter a interface limpa durante o uso
* lógica central organizada em um único loop para simplicidade

## Objetivo

Fornecer um exemplo claro de aplicação prática usando Python básico, mantendo o código minimalista, funcional e fácil de entender. O projeto é adequado para iniciantes que desejam treinar manipulação de dados no terminal.
